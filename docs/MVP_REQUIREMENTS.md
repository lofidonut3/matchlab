# MatchLab MVP 요구사항 체크리스트

## 합리적 가정 목록

### 기술적 가정
1. **인증**: 이메일+비밀번호 기반 JWT 인증 (OAuth는 V1에서 추가)
2. **데이터베이스**: SQLite로 시작, Prisma ORM 사용 (PostgreSQL 마이그레이션 용이)
3. **AI/LLM 없음**: 모든 설명 생성은 룰 기반 템플릿으로 구현
4. **실시간 채팅 없음**: MVP는 스레드형 메시지 (WebSocket은 V1)
5. **푸시 알림 없음**: 웹 알림(벨) + 이메일 알림만 구현
6. **파일 업로드 없음**: 링크 기반 포트폴리오만 지원

### 비즈니스 가정
1. **창업 MBTI**: 6문항으로 4개 성향 축 측정 (리더십/실행력/소통/리스크)
2. **매칭 스코어**: 0-100점 스케일, 가중치는 PRD 기준 (안정성 60%, 시너지 30%, 신뢰 10%)
3. **체크인 주기**: 주 2회 = 월/목 또는 화/금 고정
4. **2주 스프린트**: 14일 고정, 연장 없음 (종료 후 재시작 가능)
5. **Top10 추천**: 하드필터 통과 후 스코어 상위 10명

### 도메인 가정
1. **역할 카테고리**: 기획, 개발, 디자인, 마케팅, 운영, 기타 (6개)
2. **도메인 카테고리**: 커머스, 핀테크, 헬스케어, 교육, 콘텐츠, B2B SaaS, 기타 (7개)
3. **목표 유형**: 투자유치, 매출창출, 기술검증, 사이드프로젝트, 해커톤 (5개)

## MVP 수용 기준 체크리스트

### A. 온보딩 (60-90초)
- [ ] 이메일/비밀번호 회원가입
- [ ] 필수 입력: 주당 투입시간, 시작 가능 시점, 찾는 역할 1개
- [ ] 필수 입력: 도메인 카테고리 1개, 목표 1개
- [ ] 창업 MBTI 4-6문항 응답
- [ ] 온보딩 완료 시 즉시 추천 Top10 노출

### B. 프로필
- [ ] 프로필 조회/수정 가능
- [ ] 완성도 게이지 실시간 반영
- [ ] 필수/선택 항목 구분
- [ ] 링크(포트폴리오/깃헙/노션) 추가 가능 (선택)

### C. 추천/매칭
- [ ] 하드필터: 시작시점, 주당시간, 원격/지역, 부족역할 충족
- [ ] 스코어: 안정성(60%) + 시너지(30%) + 신뢰(10%) - 패널티
- [ ] 모든 추천 카드에 "추천 이유 2-3줄" 표시
- [ ] 모든 추천 카드에 "주의점 1개" 표시
- [ ] Top10 리스트 노출
- [ ] 후보 상세 화면에 점수 breakdown 표시

### D. 탐색
- [ ] 조건 필터(분야/커밋/역할/성향) 검색
- [ ] 차단/신고 사용자 제외

### E. 초대/메시지
- [ ] 초대 보내기/받기
- [ ] 초대 수락/거절
- [ ] 기본 메시지 기능
- [ ] 첫 대화 가이드 질문 제공

### F. 팀 생성
- [ ] 상호 수락 시 팀 자동 생성
- [ ] 팀 정보: 목표, 역할(R&R), 주간 미팅 일정
- [ ] 2주 체크리스트 자동 생성

### G. 검증 (2주 스프린트)
- [ ] 스프린트 시작/진행/종료 상태 관리
- [ ] 미션 보드(체크리스트) 표시
- [ ] 주 2회 체크인 폼 (진행률/만족도/이슈)
- [ ] 체크인 알림

### H. 리스크 감지
- [ ] 체크인 미응답 감지
- [ ] 만족도 급락 감지
- [ ] 메시지 미응답 기간 초과 감지
- [ ] 조정 제안 카드 노출

### I. 종료/피드백
- [ ] 팀 유지/해산/리매칭 선택
- [ ] 상호 피드백 입력 (약속준수/응답/기여)
- [ ] 평판/신뢰 점수 반영

### J. 알림
- [ ] 새 고적합 후보 등장 알림
- [ ] 초대/수락 알림
- [ ] 체크인 리마인더 알림
- [ ] 장기 미접속 경고

### K. 안전장치
- [ ] 차단 기능
- [ ] 신고 기능
- [ ] 미응답 경고
